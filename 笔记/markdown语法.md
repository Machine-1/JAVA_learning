# 强调效果
**粗体** 在单词或者字母间加上两个星号用来表示粗体效果
对英文字母同样有效，就像这样**app**le和apple

*斜体* 与粗体类似，但只需要加上一个星号即可

***粗斜体*** 同上，用三个星号表示
